Export Configuration

Download a single gzipped tarball containing the database, environment file, keyword lists, and Squid fragments. Keep it somewhere safe — it contains secrets (API keys, CA key paths) from brainrotfilter.env.

Download Backup (.tar.gz)
Bundle Contents
  • /var/lib/brainrotfilter/brainrotfilter.db — SQLite database (videos, channels, whitelist, settings)
  • /etc/brainrotfilter/brainrotfilter.env — environment / secrets
  • /etc/brainrotfilter/keywords.json — local keyword rules
  • /etc/brainrotfilter/community-keywords.json — synchronised community keywords
  • /etc/brainrotfilter/squid_brainrot.conf — BrainrotFilter Squid snippet (if present)
  • /etc/squid/conf.d/brainrotfilter.conf — Squid drop-in include (if present)
  • metadata.json — version, timestamp, hostname
Restore Configuration
Warning: importing a bundle will overwrite the files listed above and restart brainrotfilter.service. Any in-flight analyses will be lost. A .bak copy of each replaced file is left next to it.
